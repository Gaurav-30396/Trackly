Trackly is a simple and efficient Android app built with Jetpack Compose to help you manage the small but important tasks you perform every day. Whether it is studying, exercising, journaling, or following a routine, Trackly ensures you never forget what needs to be done.

Trackly lets you create daily tasks, track their completion, and stay consistent through clear visual progress.

## Features

### Create Daily Tasks
Add tasks you need to remember throughout the day. Trackly keeps them organized and easy to access.

### Toggle Task Completion
Each task includes a toggle switch so you can mark it as completed with a single tap.

### Circular Progress Tracker
A clean circular progress bar fills as you complete tasks, giving you instant visual feedback on your daily progress.

### Delete Tasks
Remove tasks from the list whenever needed.

### Reset All Tasks
A dedicated Reset button instantly turns off all toggles and resets the circular progress bar back to zero.

### Persistent Storage with Room
Trackly stores all tasks using Room Database, keeping them saved across app restarts.

### Built with Jetpack Compose
Modern UI components, reactive state management, and a declarative design approach.

## Tech Stack

- Kotlin
- Jetpack Compose
- Room Database
- ViewModel + StateFlow
- Material 3
- MVVM Architecture

## How Trackly Works

1. Add tasks you want to complete today.
2. Toggle a task to mark it as completed.
3. Watch the circular progress indicator update as you work through your list.
4. Delete tasks you no longer need.
5. Use the Reset button to clear all completions and start fresh.
6. All data is saved automatically through Room.
